# Dashboard — CLV & VIP Targeting

Plotly Dash application for **Project 8** (ADY201m): RFM analysis, CLV forecast, and risk-aware VIP campaign targeting on Online Retail II.

## Run locally

```bash
pip install -r requirements.txt
python dashboard/app.py
```

Open **http://127.0.0.1:8050**

## Dashboard views

| Page | Route | Purpose |
|------|-------|---------|
| Overview | `/` | Dataset KPIs, revenue trends, active customers |
| RFM Matrix | `/rfm` | RFM segmentation heatmap (SQL Q1 + Q7) |
| CLV Forecast | `/forecast` | Predicted 90-day CLV per customer |
| Campaign Targeting | `/campaign` | Top VIP list with conformal prediction intervals |

These map to the ADY201m Project 8 dashboard requirements: **RFM matrix**, **CLV forecast**, **top VIP customer list**, and **customer-level drill-down**.

## Data files

| File | Description |
|------|-------------|
| `data/customer_predictions_w3.csv` | Window-3 predictions for dashboard (committed) |
| `notebooks/outputs/*.csv` | SQL query exports (generated by `02_sql_analysis.ipynb`) |

Regenerate predictions:

```bash
python dashboard/_make_predictions.py
```
